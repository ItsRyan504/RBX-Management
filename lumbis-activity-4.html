RBXGPM

A web-based Information System for administering Roblox game pass data — players, purchases, benefits, and change audit trails.

Version 1.0.0 • April 2026

About This System

RBXGPM is a game pass administration panel developed as part of the Enterprise Development & Programming (EDP) course (Activities 4 & 5). It provides a centralized interface for monitoring players, managing passes, and generating detailed reports.

System Modules

  • Login & Authentication
  • Password Recovery (3-step flow)
  • Dashboard & Statistics Overview
  • Player Registry Management
  • Game Pass Catalog Viewer
  • Audit Log & Change History
  • Report Generator & Printer

Database Schema

  • players — registered accounts
  • gamepasses — available passes
  • benefit_types — benefit definitions
  • gamepass_benefits — pass-to-benefit map
  • player_gamepasses — ownership records
  • player_gamepasses_audit — change log

Technology Stack

HTML5 CSS3 JavaScript (ES6) MySQL / MariaDB phpMyAdmin Font Awesome 6 Google Fonts

Front-end: Pure HTML/CSS/JS • Back-end: PHP + MariaDB (planned)

Key Features

  • Credential-based login with validation
  • 3-step password recovery with OTP
  • Live player & pass search filtering
  • One-click ban / unban players
  • Add new players via modal form
  • 4 report types with print support
  • Full audit trail display
  • Dark Roblox-inspired UI theme
